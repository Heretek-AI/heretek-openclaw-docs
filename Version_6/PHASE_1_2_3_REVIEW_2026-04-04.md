#### 1. Ebbinghaus Forgetting Curve Implementation

**Mathematical Foundation:**

The Ebbinghaus forgetting curve describes exponential memory decay:

```
R(t) = S × e^(-λt)

Where:
  R(t) = Retention strength at time t
  S = Initial memory strength (importance score, 0-1)
  λ = Decay constant = ln(2) / halfLifeDays
  t = Time elapsed in days
```

| Function | Purpose | Lines |
|----------|---------|-------|
| `toDecayLambda(halfLifeDays)` | Converts half-life to decay constant | 5 |
| `calculateEbbinghausMultiplier(params)` | Calculates retention multiplier | 13 |
| `applyEbbinghausDecayToScore(params)` | Applies decay to importance score | 30 |
| `memory_retrieve(params)` | Unified retrieval with decay weighting | 49 |

| `validateImportance(importance)` | Clamps importance to 0-1 range | 12 |
| `generateMemoryId(content, createdAt)` | Generates unique memory ID | 8 |

**Memory Type Half-Lives:**

| Memory Type | Half-Life | Rationale |
|-------------|-----------|-----------|
| **Working** | 0.5 days (12 hours) | Session lifetime only |
| **Episodic** | 7 days | Recent experiences fade quickly unless reinforced |
| **Semantic** | 30 days | Facts and concepts persist longer |
| **Procedural** | 90 days | Skills and habits are long-lasting |
| **Archival** | ∞ | Permanent record, no decay applied |

#### 2. Repetition Boost Formula

Frequently accessed memories receive a logarithmic boost:

```
repetition_bonus = 1 + log10(accessCount + 1) × (repetitionBoost - 1)
```

This ensures diminishing returns — the 10th access provides less boost than the 2nd.

#### 3. Floor Protection

To prevent complete memory loss:

```
final_score = max(decayedScore, importance × floorMultiplier)
```

Default floor: 10% of original importance score.

#### 4. Configuration

```typescript
export interface EbbinghausConfig {
  enabled: boolean;           // Enable temporal decay (default: true)
  halfLifeDays: number;       // Episodic memories: 7 days
  floorMultiplier: number;    // Never decay below 10% (default: 0.1)
  repetitionBoost: number;    // Frequently accessed memories boosted (default: 1.5)
}
```

function calculateTemporalProximity(timestampA: Date, timestampB: Date, timeConstantDays: number = 7): number {

}
